# Cargo Lite Market Dashboard

Полноценный demo-дашборд для анализа рынка малотоннажных грузоперевозок в двух макрорегионах:

- Москва и Московская область
- Санкт-Петербург и Ленинградская область

Данные в проекте являются оценочными / demo data. Они смоделированы по открытой рыночной логике и не являются финансовой отчетностью компаний.

## Стек

- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- lucide-react
- SheetJS / xlsx

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Что внутри

- Главная страница со сравнением двух регионов
- Отдельные региональные срезы через переключатель
- Топ-6 компаний по каждому региону
- Финансовые, операционные, клиентские и ценовые метрики
- Расчетные индексы:
  - индекс конкурентоспособности
  - индекс операционной эффективности
  - индекс клиентского доверия
  - индекс ценовой привлекательности
  - индекс цифровой зрелости
  - общий рейтинг компании
- Проверка количества заказов через формулу `выручка / средний чек`
- Сегмент частных заказов:
  - доля частных заказов
  - количество частных заказов
  - B2B-доля
  - средний чек частного заказа
  - fit-индекс частного рынка
- KPI-карточки, таблица, bar chart, pie chart, radar chart, SWOT heatmap
- Поиск, фильтр метрик, сортировка
- Excel-экспорт с листами:
  - Summary
  - Moscow_MO
  - SPB_LO
  - Top_Companies
  - Private_Orders
  - Financial_Metrics
  - Operational_Metrics
  - SWOT
  - Recommendations

## Архитектура данных

Демонстрационный датасет находится в `src/data/companies.ts`. Его можно заменить адаптерами к реальным источникам:

- Avito
- Яндекс Карты
- 2ГИС
- сайты компаний
- отзывы
- парсинг открытых объявлений
- CRM-данные
- рекламные кабинеты

Экспорт в Excel реализован в `src/utils/exportExcel.ts`.
